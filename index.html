AUTOMATED LIGHT AND SECURITY ALARM CONTROL SYSTEM

Fusing Intelligent Lighting and Advanced Security for Seamless Living

Author 1: ASIIMWE FRED
Author 2: OKOT REAGAN

In response to the growing demand for enhanced home automation and security systems, we have developed an innovative IoT-based solution that integrates automated lighting control and security alarm functionality. This project aims to provide users with the convenience of automated lighting based on various triggers while also bolstering security through real-time alarms. This report outlines the project's methodology, implementation details, results, and future enhancements.

Security For All

The whole aim of our system is to improve community livelihood by enhancing home security plus automating lights thus saving energy bills in homes.
